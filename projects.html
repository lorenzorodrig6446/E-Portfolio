AERSP 401 — Senior Capstone  ·  Fall 2025

Autonomous Wildfire
Suppression UAV

A full systems-level design of a fully autonomous quadrotor UAV capable of detecting and suppressing wildfires without human input. The project encompassed guidance, navigation, and control (GNC) system design with a Kalman filter estimator, structural finite element analysis of the drone arms, SolidWorks CAD assembly, and test stand design — all within a strict $350 budget and 1 kg weight limit. The team was awarded 1st place in the class competition.

02
Project

AERSP 462 — Aerospace Autonomy  ·  Fall 2025

Autonomous UAV Landing
on a Moving Platform

Designed and implemented a complete guidance, navigation, and control stack in MATLAB to autonomously land a simulated quadrotor UAV on a van moving in a circular path — with no human input during flight. The cascaded PID architecture featured an outer-loop position guidance system with velocity feedforward, an inner-loop attitude controller, and a control allocation mixer mapping commands to four individual motor throttle signals. Inspired by the MBZIRC 2017 robotics competition, the simulation was tuned to minimize terminal landing error over a 150-second run.

GNC PID Control MATLAB Quadrotor Dynamics Individual Project
Role GNC Engineer (Solo)
Simulation 150 s · Final Score: 11.80

03
Project

AERSP 304 — System Dynamics & Controls  ·  Spring 2025

Funduino: DC Motor
Modeling & PID Control

A hardware-in-the-loop controls project bridging theory and physical implementation. Starting from the governing electromechanical equations, a first-order transfer function was derived analytically and then experimentally identified from step-response data collected across six input voltages via an Arduino Mega and rotary encoder. Three control architectures — open-loop, closed-loop, and PID — were designed in MATLAB simulation and implemented live on the hardware, with the PID controller achieving stable convergence to the target RPM within one second of run time.
